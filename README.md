# Face Recognition System

A Python-based **Face Recognition System** that demonstrates real-time facial recognition. Users can see how the system works behind the scenes, with recognized faces labeled with **green frames** and unrecognized faces labeled with **red brackets**. All recognized faces are stored in the **Known** folder, while unrecognized faces go to the **Unknown** folder. All credentials are securely stored in `credential.pki`.

---

## 🖥️ Features

- Real-time face detection and recognition  
- Green frames for recognized faces, red frames for unrecognized faces  
- Full process visible for learning and demonstration  
- Automatically saves recognized and unrecognized faces  
- Secure credential storage in `credential.pki`  
- All Python dependencies are already prepared in `requirements.txt`  

---

## ⚙️ Technologies Used

- **Python 3.x**  
- **OpenCV** – for face detection and image processing  
- **face_recognition** – for facial recognition and encoding  
- **NumPy** – for numerical operations  
- **Pickle / PKI** – for secure storage of credentials
